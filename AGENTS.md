# edge/AGENTS.md

## Project Overview

Edge functions that proxy Beacon Spear ingestion.

Targets:

- Cloudflare Workers
- Tencent EdgeOne

## Local Dev (Cloudflare)

From `edge/`:

- Install tooling (once):
  - `npm install`
- Run locally:
  - `npm run dev`
